Velora Coffee is an award-winning, immersive, multi-sensory concept website designed for a luxury, futuristic café brand operating in Tokyo (Ginza) and Dubai (Marina). Utilizing next-generation web technologies, it delivers cinematic storytelling, interactive 3D WebGL scenes, smooth animations, and premium glassmorphism layouts.

---

## 🌟 Key Features

* **3D Sensory Chamber:** Immersive 3D WebGL model of a floating coffee cup utilizing React Three Fiber, Drei, and Three.js. Supports scroll-driven rotation, smooth floating animations, and dynamic canvas overlay.
* **Molecular Brew Lab:** Real-time recipe customizer adjusting grind sizes (microns), volumetric extraction pressure (Bar), and gold leaf infusion (mg). Includes a dynamic SVG radar (spider) chart plotting custom flavor profiles.
* **Sovereign Lounge Registration:** High-end reservations interface allowing clients to book private sensory pods, obsidian bars, or glass solariums in Tokyo or Dubai. Generates offline digital entry tickets with active validation hash and scan-ready QR codes.
* **Audio Synthetics Control:** Integrates an ambient audio player with custom controls, providing acoustic soundscapes calibrated to enhance the sensory tasting experience.
* **Cinematic Aesthetics:** Harmonious palette of warm espresso, beige, matte black, and subtle 24K gold accents. Features responsive glassmorphism UI cards, hover effects, and premium typography.

## 🛠️ Tech Stack

* **Core Framework:** React 19, TypeScript
* **3D & Math Graphics:** Three.js, React Three Fiber (R3F), Drei
* **Styling & Theme:** Tailwind CSS v4, PostCSS, Vanilla CSS variables
* **Animations:** Motion (Framer Motion), GSAP ScrollTrigger
* **Icons:** Lucide React
* **Build Tool:** Vite

## 🚀 Getting Started

### Prerequisites
Make sure you have **Node.js** (v18+) installed on your machine.

### Installation

1. Clone or download the repository contents.
2. Install the node packages:
   ```bash
   npm install
   ```

### Run Locally

To spin up the local development server:
```bash
npm run dev
```
Open your browser and navigate to `http://localhost:3000`.

### Build for Production

To compile the production build, run:
```bash
npm run build
```
This runs TypeScript compiler checks and builds static production assets to the `dist/` directory.
